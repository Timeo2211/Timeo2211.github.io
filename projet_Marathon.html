Marathon Web

Ce projet a été en collaboration avec une autre formation présente dans mon Université. (MMI)

Le but de ce projet était de créer un site web pour une liste histoire.

Le site devait contenir une page d'accueil, une page de présentation des histoires, vous pouvez aussi vous enregistrez afin de pouvoir créer une histoire, la modifier ou la supprimer.

Ce projet a été réalisé fin décembre 2023.
